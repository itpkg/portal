Content management system 
---

### File path template

Like this:
    tmp/blogs/en-US/about-us.md

### For markdown template

File must begin with like:
    title
    ---


### For html template

File must begin with like:
    <h2>title</h2>

### For youtube video

    YOUR_VIDEO_ID.youtube

### For youku video

    YOUR_VIDEO_ID.youku
